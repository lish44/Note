# Mac 终端起别名
--------


检查已占有的名

```
alias
```

查看可用

```
type alias
```

更改 **.zshrc** 配置文件 在家目录下

语法 ： alias <新名字>=‘<老名字>' - - **=** 不能有空格
```
ex: alias ra="ranger"
```

最后 source
```
source .zshrc
```
