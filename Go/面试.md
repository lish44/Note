### array 和 slice
区别：
- array是值类型 slice是引用类型，引用的就是数组
- array拷贝的是值类型

```go
func fun() {
	a := [3]int{1, 2, 3}
	// a := []int{1, 2, 3}
	for k, v := range a {
		if k == 0 {
			a[0], a[1] = 100, 200
			fmt.Println(a)
		}
		a[k] = 100 + v
	}
	fmt.Println(a)
}
```
- 数组打印：[100,200,3] [101,102,103] v是copy过来的
- 切片打印：[100,300,3] [101,300,3] v是引用到底层array的
